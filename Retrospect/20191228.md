# 오늘 공부 한 것

- 프로그래머스 알고리즘 level 1 클리어

# 회고

생각보다 다른 짓에 시간이 많이 쓰였던 하루다.
우분투를 쓰면서 커스텀을 거의 안하다가 오늘 갑자기 터치패드부터 마우스 설정까지 하나 둘씩 하기 시작했다.
20살 이후로 계속 애플을 멀리했는데 이대로 가다간 다음 노트북은 맥북이 될것 같다.
알고리즘을 푸는 데 문제 풀이보다 오늘은 간단한 테스트 라이브러리를 만들면서 했는데 이로 인한 시간이 더 많이 걸렸다. JUnit과 같은 테스트 라이브러리를 사용해 볼까 하다가 그냥 직접 만들어 보고 싶기도 하고 해서(사실은 배우기 귀찮아서) 내가 필요한 기능만 구현했다. 그러다 내가 Java에 대해 정말 아는 것이 없음을 느꼈다. 만약 내가 백엔드 개발로(하고 싶지만!) 취업이 결정나게 된다면 Java는 피할 수 없을 텐데 부스트 캠프를 통해 Javascript를 공부한 만큼 Java에도 다시 시간을 꽤나 많이 들여야 겠다는 생각이 들었다. 지금은 알고리즘만 끄적거리다 보니 Java를 쓰지만 Object 클래스나 Generic, 람다와 같은 문법은 거의 사용할 일이 없었다. 당장 면접이 코앞이라는 좋은 핑계로 또 배우는 것을 미루고 있는데 정말 빠른 시일 내에 제대로 공부해 봐야 겠다는 생각이 들었다. 아직은 꿈같은 이야기지만 정말 운좋게 부스트 캠프를 통해 지원하는 네이버 및 네이버 계열사는 아마 다 입사하면 spring을 또 새롭게 배워서 시작할텐데 부스트 캠프를 하며 아쉬웠던 점이 또 반복되지 않게 준비도 미리 많이 해서 새롭게 시작하고 싶다.
